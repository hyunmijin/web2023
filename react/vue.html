vue

webpack은 JavaScript 애플리케이션을 위한 정적 모듈 번들러입니다.

Vue.js는 사용자 인터페이스를 구축하기 위한 프로그레시브 프레임워크입니다.
프로그레시브 프레임워크는 애플리케이션의 규모와 요구 사항에 따라 적용할 수 있는 기능을 단계적으로 적용할 수 있는 방식을 의미합니다.
Vue.js는 단일 페이지 애플리케이션(SPA) 개발에 많이 사용되며, 웹 애플리케이션의 사용자 인터페이스를 구성하기 위해 HTML, CSS 및 JavaScript를 사용합니다.

    mode: 'development', //development, production(개발모드)

    },
    output: {

        filename: 'js/[name].js'

}

1.6 HTML 번들러 설정

npm install html-webpack-plugin --save-dev
"devDependencies": {
    "html-webpack-plugin": "^5.5.1",
    "webpack": "^5.83.1",
    "webpack-cli": "^5.1.1",
    "webpack-dev-server": "^4.15.0"
}

const path = require('path');
const HtmlWebpackPlugin = require('html-webpack-plugin');

module.exports = {
    mode: 'production', //development, production(개발모드)
    entry: {
        index: './src/index.js',
    },
    output: {
        path: path.resolve(__dirname, 'public'),
        filename: 'js/[name].js',
    },
    plugins: [new HtmlWebpackPlugin({
        template: './src/index.html',   //번들하기 전
        filename: './index.html',        //번들 후
        hash: true,
        showErrors: true
    })],
}

vite

Vite는 최신 웹 개발을 위한 빠르고 가벼운 빌드 도구입니다. 웹 애플리케이션의 개발 및 빌드 과정을 간소화하고 개발 서버를 제공하여 실시간 모듈 번들링을 지원합니다.

Webpack은 모듈 번들러(module bundler)로, 웹 애플리케이션을 구성하는 다양한 모듈들을 하나의 번들(bundle)로 만들어줍니다. 웹 애플리케이션은 일반적으로 여러 개의 자바스크립트 파일, CSS 파일, 이미지 파일 등 다양한 종류의 파일로 구성되는데, 이러한 파일들을 하나의 번들로 묶어줌으로써 애플리케이션의 성능을 향상시킬 수 있습니다.

Webpack은 모듈 시스템을 사용하는 자바스크립트 애플리케이션에 특히 유용합니다. 모듈 시스템은 코드를 모듈 단위로 분리하여 개발하고, 필요한 모듈들을 가져와서 재사용할 수 있는 방법을 제공합니다. 이는 코드의 유지보수성과 확장성을 향상시키는 데 도움이 됩니다.

Webpack은 주어진 애플리케이션에 필요한 모든 모듈을 분석하고, 이를 의존성 그래프(dependency graph)로 만듭니다. 의존성 그래프는 각 모듈 간의 종속 관계를 보여주는 구조입니다. 그리고 웹팩은 이 그래프를 기반으로 모든 모듈을 하나의 번들로 묶습니다. 번들링된 결과물은 최적화되어 서버에 배포할 수 있습니다.

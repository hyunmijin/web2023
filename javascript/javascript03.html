01. 선언적 함수

함수를 호출할 때 함수의 앞 또는 뒤에서 호출이 가능합니다.

{
    function func(){
        document.write("실행되었습니다.
");   //실행문이 있어야함
    }
    func();                                                      //가장 기본적인 함수
}
